# ✨ Neon Tic Tac Toe – Modern AI Web Game

A **modern neon-styled Tic Tac Toe game** built with **HTML, CSS, and Vanilla JavaScript**.

This project features **AI vs Human gameplay, Player vs Player mode, multiple AI difficulty levels, animated neon UI, and real-time scoreboard tracking**.

The entire game is implemented using **core web technologies without any external libraries or frameworks**.

# 📌 Project Overview

This project is a **visually enhanced and feature-rich version of the classic Tic Tac Toe game**.

The goal was to build a game that combines:

- Clean and maintainable JavaScript logic
- Smooth neon-themed UI animations
- Multiple gameplay modes
- Intelligent AI using the **Minimax algorithm**

Unlike simple Tic Tac Toe implementations, this version focuses on **user experience, animation design, and AI gameplay mechanics**.

---

# 🎮 Game Modes

## 🤖 AI vs Human

Play against the computer with adjustable difficulty levels.

AI difficulty levels:

- Easy — Random moves
- Medium — Strategic moves with blocking and occasional optimal plays
- Hard — Perfect AI using the **Minimax algorithm**

Players can also choose their symbol:

- X
- O

---

## 👥 Player vs Player

Two players can play locally on the same device.

- Player 1 uses **X**
- Player 2 uses **O**
- Turns alternate automatically.

# 🧠 Game Logic

### Turn System

Players take turns placing symbols on the board.

The game checks after each move for:

- Winning combinations
- Draw conditions

---

### Winning Conditions

A player wins when they complete one of these combinations:

```
Rows
Columns
Diagonals
```

Example combinations:

```
0 1 2
3 4 5
6 7 8
```

```
0 3 6
1 4 7
2 5 8
```

```
0 4 8
2 4 6
```

---

### Minimax AI (Hard Mode)

The **Hard AI difficulty uses the Minimax algorithm**, which evaluates all possible game states to determine the optimal move.

The algorithm:

1. Simulates all possible moves
2. Evaluates win/loss outcomes
3. Selects the move with the highest score

This makes the AI extremely strong.
